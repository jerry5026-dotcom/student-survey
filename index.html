📖 사용 방법 안내

🚀 1. 시작하기

  1. Google 계정으로 로그인 — 메인 화면에서 Google 계정 로그인 버튼을 클릭합니다.
  2. 처음 사용한다면 Google Cloud Console에서 Google Forms API와 Google Drive API를 활성화해야 합니다 (로그인 화면 하단의 ⚙️ 설정 안내 참고).

📝 2. 질문 관리 탭

  1. 기본 질문 39개가 로드되어 있습니다 — 고등학생 학기 초 기초조사에 최적화된 구성입니다.
  2. + 질문 추가 — 새 질문을 직접 추가합니다. 서술형/단일선택/복수선택 지원.
  3. 드래그로 순서 변경 — 행 왼쪽의 ⋮⋮ 핸들을 잡고 끌어올리거나 내려 순서를 바꿉니다.
  4. ⭐ 내 기본값으로 저장 — 지금 설정한 질문 목록을 나만의 기본값으로 저장합니다.
  5. 🔄 기본값으로 초기화 — 저장한 나만의 기본값(없으면 공장 초기값)으로 돌립니다.
  6. 📥 질문 엑셀 다운 / 📤 업로드 — 질문 목록을 엑셀로 받아 오프라인에서 수정하고, 다시 업로드해 반영할 수 있습니다.
💡 업로드 엑셀 양식은 다운로드한 파일과 동일한 헤더(번호 / 질문 / 선택지 / 유형 / 필수)를 유지해야 합니다.

📋 3. 폼 생성 탭

  1. 📋 Google Form 만들기 버튼을 누르면 현재 질문 목록으로 구글 폼이 자동 생성됩니다.
  2. 학생 배포 링크를 복사해 학생들에게 공유하세요.
  3. 교사 수정 링크로는 폼을 직접 수정할 수 있습니다.

👤 4. 학생 응답 탭

  1. 🔄 응답 새로고침으로 학생 응답을 불러옵니다.
  2. 학생을 선택하면 카테고리별(진로·가족·학습·성격 등)로 정리된 응답을 확인합니다.
  3. 📊 엑셀 다운로드로 전체 응답을 다운받습니다:
    • 시트1 학생조회 — B3에 학생 이름을 입력하면 해당 학생의 응답이 수식으로 자동 표시
    • 시트2 전체응답 — 제출 시각 순으로 정렬된 원본 데이터
    • AI 상담 질문을 생성한 경우, D5 셀부터 해당 학생의 AI 질문이 자동으로 표시됩니다

🤖 5. AI 상담 질문 탭

  1. 먼저 ⚙️ AI 설정 탭에서 사용할 AI(OpenAI/Claude/Gemini) 모델과 API 키를 설정합니다.
  2. Gemini 3 Flash / Flash Lite는 무료(1,500회/일)로 사용할 수 있어 입문자에게 권장합니다.
  3. ✨ 선택 학생 생성 — 학생 응답 탭에서 선택한 학생에 대해 개별 질문 생성
  4. 👥 전체 학생 일괄 생성 — 응답한 모든 학생에 대해 차례로 AI 질문 생성 (실패 건은 건너뜀)
  5. 생성된 질문은 자동 저장되며, 다시 접속해도 엑셀에 포함됩니다.
⚠️ 일괄 생성은 학생 수 × API 호출이 발생하므로 유료 모델의 경우 비용이 발생합니다. 모델을 바꾸면 해당 모델 한도 내에서 다시 호출됩니다.

⚙️ 6. AI 설정 탭

  1. 제공사 선택 → 모델 선택 → API 키 입력 3단계 순서로 진행합니다.
  2. API 키 입력 후 유효성 확인을 누르거나, 붙여넣기 하면 자동으로 검증됩니다.
  3. 여러 제공사의 키를 모두 등록하면 AI 탭 드롭다운에서 자유롭게 전환할 수 있습니다.
